# Exercise 2: E-commerce Platform Search Function

## Scenario (Cognizant DN5.0 Handbook)
You are working on the search functionality of an e-commerce platform. The
search needs to be optimized for fast performance.

## Requirements Implemented
- [x] `Product` class with `productId`, `productName`, `category`
- [x] Linear search implemented, run on a plain array
- [x] Binary search implemented, run on a sorted array
- [x] Analysis of time complexity (Big O) for both algorithms

## How to Run
1. Open this folder in VS Code.
2. Open `src/ProductSearchDemo.java`.
3. Click **Run** above `main` (or `Ctrl+F5`).
